# 🎆 FireScreen - 动画屏幕保护

精美的动画屏幕保护应用，支持密码锁定和全屏播放。

## ✨ 功能特点

- 🎆 **精美烟花动画** - 五彩缤纷的烟花从零开始绽放
- 🔒 **密码保护** - 设置4位数密码锁定屏幕
- 🔓 **安全解锁** - 任意键盘或鼠标触发解锁界面
- 🌟 **全屏播放** - 沉浸式动画体验
- 🔑 **万能密码** - 0000 可解锁（防止忘记密码）
- 🎨 **现代化设计** - 精致的UI界面

## 🚀 使用方法

### 1. 选择动画
在首页选择您喜欢的动画效果（目前支持烟花动画）

### 2. 设置密码
点击"应用"按钮，在弹窗中输入4位数字密码

### 3. 全屏播放
确认密码后，动画将全屏播放

### 4. 解锁退出
移动鼠标或按下任意键，输入密码解锁（万能密码：0000）

## 🎯 密码说明

- 用户密码：您自己设置的4位数字
- 万能密码：`0000`（确保您不会被锁在外面）
